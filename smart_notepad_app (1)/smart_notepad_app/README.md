# Smart Notepad - Java Swing

A beautiful Java Notepad application with a unique smart feature.

## Features

- Save files
- Open files
- Clear editor
- Dark mode
- Word counter
- Character counter

## UNIQUE FEATURE

Smart Emoji Replacer

Automatically converts words into emojis:

- happy -> 😊
- sad -> 😢
- love -> ❤️
- cool -> 😎

Example:
```text
I am happy
```

Becomes:
```text
I am 😊
```

## Run

Compile:
```bash
javac src/notepad/SmartNotepad.java
```

Run:
```bash
java notepad.SmartNotepad
```
